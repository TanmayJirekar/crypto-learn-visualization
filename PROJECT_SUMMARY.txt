HOW TO RUN:
================================================================================

OPTION 1: Direct Opening
- Double-click index.html
- OR right-click → Open with → Your browser

OPTION 2: Python Local Server
$ cd /vercel/share/v0-project
$ python3 -m http.server 8000
Then visit: http://localhost:8000

OPTION 3: Node.js HTTP Server
$ npx http-server
Then visit: http://localhost:8080

OPTION 4: VS Code Live Server
- Install "Live Server" extension
- Right-click index.html
- Select "Open with Live Server"

Stored Data:

SECURITY NOTES:
================================================================================

IMPORTANT:
- This is an EDUCATIONAL application
- Algorithms shown are for LEARNING purposes only
- DO NOT use for real cryptographic security
- For production security, use industry libraries

Examples of what's safe:
✓ Understanding how encryption works
✓ Learning algorithm concepts
✓ Educational demonstrations
✓ Quiz and learning platform

Examples of what's NOT safe:
✗ Encrypting sensitive passwords
✗ Securing financial information
✗ Production cryptography
✗ Storing real secrets

For real security, use:
- NaCl.js (libsodium bindings)
- TweetNaCl.js
- bcrypt for passwords
- Professional SSL/TLS libraries
